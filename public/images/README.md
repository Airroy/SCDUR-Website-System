# Header Banner Image

วางไฟล์รูปภาพชื่อ `header-banner.jpg` ในโฟลเดอร์นี้

## ข้อกำหนด:
- ชื่อไฟล์: `header-banner.jpg`
- ขนาดที่แนะนำ: 1920x200 pixels (สี่เหลี่ยมผืนผ้า)
- รูปแบบไฟล์: JPG, PNG

## วิธีเปลี่ยนรูป:
1. เตรียมรูปภาพที่ต้องการ
2. เปลี่ยนชื่อเป็น `header-banner.jpg`
3. วางทับไฟล์เดิมในโฟลเดอร์ `public/images/`
4. รีเฟรชหน้าเว็บเพื่อดูผลลัพธ์

หากต้องการใช้รูปชื่ออื่น สามารถแก้ไขที่ไฟล์:
`resources/views/components/layouts/frontend.blade.php` บรรทัดที่มี `header-banner.jpg`
